Learning WebComponents

A web komponensek csak és kizárólag saját DOM fájuk felépítéséért és stílusozásáért felelnek, ezek viszont kívülről nem módosíthatók.

Az elkészített web komponenseket bármilyen más web alkalmazásban felhasználhatjuk, használt JavaScript könyvtártól, és keretrendszertől függetlenül

Példák, hogy milyen alkalmazásban használhatod újra:
  • React
  • Angular
  • Vue
  • ...
